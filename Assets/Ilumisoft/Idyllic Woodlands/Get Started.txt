Setup for the Built-In Render Pipeline
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
By default, the asset pack is configured for the URP (Universal Render Pipeline). To set it up for the Built-In Rneder Pipeline, please install 
the following packages from the Package Manager:

- Post Processing 
- Shader Graph

To install the packages, click Window->Package Manager and select "Packages: Unity Registry" in the top menu. Select each package and click the install button.

After that double click the Assets/Ilumisoft/Idyllic Woodlands/Idyllic Woodlands Builtin.unitypackage and select import.
This will overwrite all shaders, materials and demo assets to work properly in a URP project.
